Introduction:


APICAM is designed for your machine learning project which uses camera for object detection,face detection,mask detection etc. APICAM shares your mobile camera with your destination server and receive data. Received data is processed and displays rectangle with titles.

Help:


Before starting you have to add your server location and port number. For best result use your local IP address and set video quality to low.
Received data parameter:
1.PosX --> X coordinates
2.PosY --> Y coordinates
3.Height --> Height of rectangle
4.Width --> Width of rectangle
5.ID --> ID of object
6. Result --> Result of Image
7.Accuracy --> Accuracy of Result
For server you need python code which creates API on server.
You can get more details and python code on website "https://diptsoft.com/app/apicam".

Privacy:


APICAM requires following permission
1. Camera
For capturing video, which will send to your desired server location.
2. Internet
Intenet/wifi helps to share video data to your server.
